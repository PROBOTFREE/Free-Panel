# Security Policy

## Supported Versions

The following versions of **FREE PANEL** currently receive security updates.

| Version        | Supported |
| -------------- | --------- |
| Latest Release | ✅         |
| Older Versions | ❌         |

We recommend always using the **latest release** to ensure security fixes and improvements.

## Reporting a Vulnerability

If you discover a security vulnerability in **FREE PANEL**, please report it privately.

Please **do not open a public GitHub issue** for security vulnerabilities.

Instead, report it by:

* Opening a **GitHub Security Advisory**
* Or contacting the maintainer directly

Please include the following information:

* A description of the vulnerability
* Steps to reproduce the issue
* Potential impact

We will investigate the issue and respond as soon as possible.

Thank you for helping keep **FREE PANEL** secure.
